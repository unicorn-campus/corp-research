## 2. 사업 개요 및 핵심 기술

### 핵심 기술: VIA PUF™

반도체 제조 공정의 미세 공정 편차를 활용하여 복제 불가능한 하드웨어 보안 키를 생성하는 기술.  
별도 저장 공간이 불필요하며 국제 특허 150개 이상 보유.

### 주요 제품

| 제품 | 설명 |
|------|------|
| PUF IP | VIA PUF 기술 라이선스 공급 |
| PUF 보안칩 (STR) | Xbox 디바이스 정품 인증용 보안칩 |
| Giant 5 (G5) | PUF+PQC 결합 차세대 칩, LG유플러스 공급 |
| PQC PUF-eSIM | LG유플러스 협력, 세계 최초 상용화 (2025.04) |
| PQC KMS | 양자내성암호 기반 키 관리 솔루션 |

### 주요 고객사

| 고객사 | 협력 내용 |
|--------|-----------|
| Microsoft | Xbox 디바이스 액세서리 보안칩 'STR' 초도 납품 (2026.05, 4년 프로젝트 완성) |
| LG유플러스 | G5 칩 공급, PQC PUF-eSIM 공동개발, PQC KMS 공동개발, MOU 체결 |
| KT | 양자보안 국책과제(70억, 2026.04~2029.12) 수요기업 참여 |

_출처: 웹 검색 (이데일리·지디넷코리아·프라임경제·비즈니스포스트 등)_

## 8. SWOT 종합 評價

| 구분 | 내용 |
|------|------|
| **강점 (S)** | VIA PUF™ 세계 최초 상용화 + 150개 이상 국제 특허 / 글로벌 빅테크(MS) 밸류체인 진입 및 통신사 레퍼런스 / IPO 후 재무 안정화 (부채비율 6.47%, 유동비율 2,988%) |
| **약점 (W)** | 지속 영업손실 (2024년 영업이익률 △100%, 누적 결손금 382억) / 매출 성장 둔화 (2023년 +141% → 2024년 +8%) / 과도한 CB 발행으로 주주 희석 리스크 |
| **기회 (O)** | SKT 해킹 이후 하드웨어 보안 수요 급증·정책 규제 강화 / 양자컴퓨터 위협 가속 → PQC+PUF 수요 폭발 전망 / MS 납품 본격화 및 통신사 커버리지 확대 |
| **위협 (T)** | 글로벌 대기업(Infineon·NXP) 대비 규모·자원 열위 / CB 전환 지속에 따른 주가 희석 압력 / 적자 지속 시 자금 조달 어려움 및 R&D 투자 제약 |

**한 줄 총평**: PUF 기술 독보성과 글로벌 레퍼런스는 확보했으나, 수익화 속도가 관건인 기술특례 스타트업 단계 기업.
